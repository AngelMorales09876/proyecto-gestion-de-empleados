#Descripción del Proyecto

Este proyecto tiene como finalidad almacenar la información básica de los empleados, incluyendo datos como ID, nombre, apellido y otros campos adicionales.

Su funcionalidad principal consiste en que, al completar los campos del formulario, la información ingresada se pueda almacenar en un archivo txt.

De esta manera, el sistema facilita el registro, organización y respaldo de la información de los empleados de forma práctica y ordenada.

Tambien permite cargar la informacion del archivo txt a la aplicacion.
